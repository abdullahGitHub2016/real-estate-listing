# 🏠 DreamHome BD - Real Estate Marketplace

DreamHome BD is a modern, high-performance real estate portal designed for the Bangladesh property market.  
It provides a seamless interface for buyers, renters, and developers to connect, featuring a light, airy UI with a professional **Navy and Green aesthetic**.

---

## 🚀 Features

- **Advanced Search Hero:** High-impact search interface with location and category filtering.
- **Featured Listings:** 3-column responsive grid showcasing properties with 10+ specific feature icons (Beds, Baths, Size, Lift, etc.).
- **Developer Directory:** A dedicated, perfectly centered directory for Platinum Developers with property count tracking.
- **Responsive Design:** Optimized for Mobile, Tablet, and Desktop using Tailwind CSS.
- **Interactive UI:** Hover effects, sticky navigation, and professional glassmorphism elements.

---

## 🛠️ Tech Stack

**Frontend**
- HTML5
- Tailwind CSS (via CDN / CLI)

- [Inter Google Font](https://fonts.google.com/specimen/Inter)

## ⚙️ Installation & Setup

### Clone the repository

```bash
git clone https://github.com/your-username/dreamhome-bd.git
```

### Navigate to the project folder

```bash
cd dreamhome-bd
```

### Open the project

Simply open **index.html** in any modern web browser.  
No compilation is required as it uses **Tailwind via CDN** for development.
